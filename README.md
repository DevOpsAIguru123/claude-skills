# claude-skills

Personal Claude Code skills plugin — stock analysis and more.

## Install

```bash
claude plugin marketplace add github:DevOpsAIguru123/claude-skills
claude plugin install stock-analysis@claude-skills
```

## Skills

### stock-analysis

Comprehensive stock research with a **Buy / Hold / Sell** signal.

- **No API key needed** — price, P/E, EPS, market cap, analyst targets via Yahoo Finance
- **Optional free key** — RSI-14, MACD, SMA-50 via [Alpha Vantage](https://www.alphavantage.co/support/#api-key)

**Usage:** Ask Claude: *"Analyze AAPL"*, *"Should I buy Tesla?"*, *"Is Nvidia overvalued?"*
